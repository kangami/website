About Me

Hi There, I'm Karim a Software Engineer based in Toronto, I worked at 3 Ways International Logistics as Computer Engineer.

I discovered computers at the age of 16 and since then, I have to my credit three brand new machines that I spoil because at my beginnings I was always tinkering with my PCs which earned me spend a year without PC because my parents were fed up. Since this very hard period I have never spoiled a PC and all my computers that followed this event are still functional. Thanks to my nonsense I have a deep knowledge of the computer tool so a real concern for the memory occupied by my data structures and the speed of the codes that I execute. In a team I am the one who always cares about the respect of good practices and standards even if the code is functional; then the principles such as DRY (Don't Repeat Yourself), KISS ('Keep it Simple Stupid'), SOLID I force myself to respect them on my daily basis.

Having experienced several languages like php, C #, Java SE, Java EE etc but today I feel more comfortable with JavaScript because in a word it is the future; There is something for everyone you want a real-time application NodeJs-socket.io, you want a purely native app on the main os ios and android ReactNative, you want to develop a microservice architecture SenecaJS. So I really want to master it.
